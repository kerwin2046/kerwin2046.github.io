作为全栈工程师，既要写代码又要和产品、设计协作，工具选型会直接影响效率。这篇从实际使用出发，聊聊 2023 年我常用的设计/协作工具和取舍。

## 设计：Figma 为主

UI 设计我们团队以 Figma 为主，组件库、设计规范、动效都能在一个文件里维护，开发用 Dev Mode 或插件就能拿到标注和 Token，减少来回对稿。

## 协作：Linear + Notion

需求与 Bug 用 Linear 管理，文档和知识库用 Notion。两者都能和 GitHub / Slack 打通，迭代节奏清晰，也方便事后复盘。

## 小结

没有银弹，按团队规模和习惯选「够用且能打通」的一套即可；重点是约定好流程，而不是堆工具。
